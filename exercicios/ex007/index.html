Significado de semântica

Ramo da linguística que estuda a significação das palavras e suas mudanças de sentido ao longo do tempo, bem como a representação do sentido dos enunciados.

O significado dos vocábulos, por oposição à sua forma.

Para tudo que é forma se usa folha de estilo ou CSS

Para tudo que é significado se usa semântica, ou HTML5

HTML5 é a linguagem de marcação que usamos para estruturar e dar significado para o conteúdo de uma página web.
